# MSPWeChat
## 介绍
这是一个我个人仿写微信iOS客户端的项目。
因为最初的代码是我刚学习iOS开发3个月时所写，乱七八糟不忍直视。但至少还有那么点可利用的地方，所以我开始了浩浩荡荡的重写之路。
因为我水平有限，不会写接口，所以所有与网络打交道的数据都打算用本地的来替代。
出于学习的目的，会列出想要实现的功能列表，并对已完成的做出标记。
ps:已经被移除的文件都未删除，所以有很多不在project中的无用的文件，说不定会有参考价值
